1. Abrir QGIS.

2. Ir a:
   Capa -> Anadir capa -> Anadir capa WFS/OGC API - Features

3. Crear una conexion nueva:
   Nombre: Helsinki WFS
   URL: https://kartta.hel.fi/ws/geoserver/avoindata/wfs?request=getCapabilities

4. Tocar Conectar.

5. Agregar capa de espacios verdes:
   Buscar: viher
   Seleccionar: 1-Viheralue / YLRE_Viheralue_alue
   Tocar Anadir

6. Agregar capa de parcelas:
   Buscar: kiinteisto
   Seleccionar: 2-Kiinteisto / Kiinteisto_alue
   Tocar Anadir

7. Verificar que ambas capas tengan datos:
   Click derecho sobre la capa -> Abrir tabla de atributos

8. Antes de exportar, crear el schema analysis si no existe.
   En DB Manager -> SQL Window ejecutar:

   CREATE SCHEMA IF NOT EXISTS analysis;

9. Exportar YLRE_Viheralue_alue a PostGIS:
   DB Manager -> Export to PostgreSQL

10. Configurar asi:
    Database: citydb_test
    Schema: analysis
    Table: green_areas
    Primary key: id
    Geometry column: geom
    Source SRID: EPSG:3879
    Target SRID: EPSG:4326
    Encoding: Automatic

11. Marcar:
    [x] Replace destination table
    [x] Convert field names to lowercase
    [x] Create spatial index

12. Guardar / Exportar.

13. Exportar Kiinteisto_alue a PostGIS:
    DB Manager -> Export to PostgreSQL

14. Configurar asi:
    Database: citydb_test
    Schema: analysis
    Table: parcels
    Primary key: id
    Geometry column: geom
    Source SRID: EPSG:3879
    Target SRID: EPSG:4326
    Encoding: Automatic

15. Marcar:
    [x] Replace destination table
    [x] Convert field names to lowercase
    [x] Create spatial index

16. Guardar / Exportar.

17. Verificar que quedaron cargadas en PostGIS:

    SELECT COUNT(*)
    FROM analysis.green_areas;

    SELECT COUNT(*)
    FROM analysis.parcels;
