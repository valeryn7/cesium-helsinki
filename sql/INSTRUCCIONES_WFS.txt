1. Abrir QGIS.

2. Ir a:
   Capa → Añadir capa → Añadir capa WFS/OGC API - Features

3. Crear una conexión nueva:
   Nombre: Helsinki WFS
   URL: https://kartta.hel.fi/ws/geoserver/avoindata/wfs?request=getCapabilities

4. Tocar Conectar.

5. En el buscador escribir:
   viher

6. Buscar y seleccionar una capa que contenga:
   Viheralue
   Por ejemplo:
   YLRE_Viheralue_alue

7. Tocar Añadir y esperar a que cargue en el mapa.

8. Verificar que la capa tenga datos:
   Click derecho sobre la capa → Abrir tabla de atributos

9. Antes de exportar, crear el schema analysis si no existe.
   En DB Manager → SQL Window ejecutar:

   CREATE SCHEMA IF NOT EXISTS analysis;

10. Exportar la capa a PostGIS:
    DB Manager → Export to PostgreSQL

11. Configurar así:
    Database: citydb_test
    Schema: analysis
    Table: green_areas
    Primary key: id
    Geometry column: geom
    Source SRID: EPSG:3879
    Target SRID: EPSG:4326
    Encoding: Automatic

12. Marcar:
    [x] Replace destination table
    [x] Convert field names to lowercase
    [x] Create spatial index

13. Guardar / Exportar.

14. Verificar que quedó cargada en PostGIS:

    SELECT COUNT(*)
    FROM analysis.green_areas;
